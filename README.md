## 프로젝트 소개
<p align="center"><img src="https://user-images.githubusercontent.com/78775910/170033090-e4ce4cf8-7a05-4d6d-a664-e8ac9c89104b.png" width=30%></p>

<p align="center"><img src="https://user-images.githubusercontent.com/78775910/170033136-c2f3893d-35e5-4aa8-86c6-5b5367b9b27f.png" width=30%></p>
>**`라이어게임`** 은 **`거짓말쟁이를 찾는`** 게임입니다.
>
>한 방에 있는 사람 중 라이어를 제외한 참여자는 제시어 카드를 받고 해당 제시어를 설명합니다. 
>
>참여자는 라이어에게 제시어를 들키지 않게 설명하고, 라이어는 제시어 카드를 맞춰야 하는 모바일 뷰 기반 게임 웹사이트입니다.
